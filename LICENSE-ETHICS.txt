# 人格語言倫理附加條款（Addendum to Apache 2.0 License）
Version 1.0

本附加條款適用於 NewCoolOwl（新酷鷹）及其衍生作品，並具有與主授權同等之法律效力。

本附加條款 **並不限制個人、研究與非營利使用**。
其目的是保護使用者之語言人格，不被模仿、剝削或商品化利用。

------------------------------------------------------------
1. 不得儲存或收集可逆還原之原始語句。
2. 不得建立、訓練或散布旨在「模仿特定個人語氣 / 人格」之模型。
3. 不得用於心理操控、情緒影響、誘導式溝通或行為干預。
4. 使用者之語言資料與語言風格特徵視為「人格資料」並屬於使用者本人。
5. 若用於商業用途，必須先通過「人格語言倫理審查」並取得作者同意。
6. 若商業行為後違反本條款，作者有權撤回相關授權。
------------------------------------------------------------

本附加條款適用於所有：
- 二次開發
- 商業整合
- API / SDK / 模型嵌入
- 雲端部署或服務化使用

個人本地使用不受此條款限制。

「語言屬於說話者。
人格不可被複製、販售或操控。」
